# Web Traffic Analytics

## 📊 Project Overview
This project focuses on analyzing website traffic data to understand user behavior and engagement. The dashboard helps identify important metrics like page views, bounce rate, traffic sources, and user sessions.

## 🛠️ Tools & Technologies Used
- Power BI (for visualization)
- Microsoft Excel (dataset)
- DAX (for calculations)

## 🔧 Work Done in This Project
- Imported and cleaned web traffic dataset
- Transformed data using Power Query
- Created measures like total users, sessions, and bounce rate
- Designed interactive dashboard with filters and slicers
- Built visualizations like line charts, bar charts, and pie charts
- Analyzed user behavior and traffic sources

## 📌 Key Features
- Traffic source analysis (Direct, Social, Referral, etc.)
- Page-wise performance
- User sessions and page views tracking
- Bounce rate analysis
- Interactive filters (slicers)

## 📈 Insights Gained
- Identified major traffic sources
- Found pages with highest user engagement
- Analyzed bounce rate to detect user drop-off
- Observed user activity trends over time

## 🎯 Objective
To analyze website traffic data and understand user behavior to improve website performance and user experience.

- Web Analytics Concepts
- Data Visualization
- DAX Calculations
- User Behavior Analysis
